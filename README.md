# 推金币——技术要点

- 这是一款益智休闲游戏，通过点击掉落金币，使金币从正前方掉落，获得更多的金币。

## 运行环境
- 引擎版本：Cocos Creator 3.8.0
- 编程语言：TypeScript

### 游戏类型

- 休闲，益智，3D，安卓，H5；

## 游戏玩法

一款益智休闲游戏，通过点击掉落金币，使金币从正前方掉落，获得更多的金币。若金币从两侧掉落，则无效。
